Массивы

В этом уроке мы поговорим, что такое массив, зачем он нужен и как с ним работать.

Обычный массив

Массив - это структура данных, которая позволяет хранить множество элементов одного типа под одним именем и обращаться к ним по индексу. Чтобы создать массив, мы должны написать его тип данных, его имя, и "= new int[];". Вместо int вы должны написать тип данных, который у вас указан в начале. Также после названия или типа данных нужно написать квадратные скобки, и в них указать количество элементов, которые будут у вас в массиве. Давайте создадим простой массив целых чисел: 

public class Main {
    public static void main(String[] args) {
        int[] numbers = new int[4];
    }
}
Теперь у нас есть массив, который содержит 4 элемента. Но на данный момент все элементы пустые. Чтобы дать какому-то элементу значение, мы должны написать название массива, и в квадратных скобках указать индекс элемента и через оператор "=" присвоить значение. Но если мы запишем вместо индекса 1, то заменится второй элемент, так как нумерация массивов в Java начинается с 0, а не 1. То есть: - первый элемент - индекс 0; - второй элемент - индекс 1; - третий элемент - индекс 2; - четвертый элемент - индекс 3; и так далее... Чтобы воспользоваться каким-либо элементом, например, присвоить в переменную значение из массива, мы просто пишем название массива, и в скобках пишем индекс элемента. Давайте применим полученные знания на практике: 

package your.package;

public class Main {
    public static void main(String[] args) {
        int[] nums = new int[2];
        
        nums[0] = 2;
        nums[1] = 9;
        
        int result = nums[0] + nums[1];
        
        System.out.print(result);
    }
}
Но также мы можем сразу при создании массива указать все значения, для этого вместо "new int[3];" пишем {76, 3, -46}, и так мы сразу укажем все значения.

Многомерный массив

В массивы можно также поместить другой массив, и он уже будет многомерным. Для этого в начале, где ставим [], мы должны написать [][], таким образом мы скажем программе, что это многомерный массив, и указывать нужно столько, сколько у вас подмассивов. Если 2, то добавляете к [] - [][], если 3, то [][][]; но их очень редко используют, обычно используют двумерные и обычные. Двумерный массив — это тот массив, который содержит 1 подмассив. Давайте создадим двумерный массив: 

package your.package;

public class Main {
    public static void main(String[] args) {
        int[][] nums = {{19, 43, 43}, {-32, -1, -100}};
        
        System.out.print(nums[0][1]);
    }
}
Тут мы создали двумерный массив и вывели из первого подмассива элемент по индексу 1. То есть в первых скобках указываем индекс подмассива, а во вторых скобках указываем элемент из этого подмассива. И по сути можно делать хоть многомерный массив, где в будет по 100 подмассивов.
